Grapher v1.1

  • Added a presets menu for saving and loading graph parameters
  • Added the ability to plot a 3D parametric curve rather than a surface
  • Consolidated versions for all platforms into one jar file
  • Made the program check for updates
  • Pressing enter after typing in an expression now updates the graph.
  • Changed the default viewing angle of 3D graphs
  • Fixed 3D axes colors: Red=x, Green=y, Blue=z
  • Made graph shading subtly more exaggerated
  • Increased default window size
  • Removed 3D graph panning, since it was difficult to pick the right place. Coordinates need to be entered in the bounds window.
  • Moved the Cross Section menu in the cross section graph into the Graph menu.
  • Miscellaneous improvements
